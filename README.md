# moon

A tiny project for exploring lunar phases and celestial events.

## Features

- Track moon phases
- Calculate lunar events
- Simple CLI interface

## Installation

```bash
pm install -g moon
```

## Usage

```bash
moon phase
moon event --date 2026-06-16
```
