# moon

A tiny project to explore lunar phases and celestial events.

## Features

- Track moon phases
- Calculate eclipse dates
- Generate star maps

## Installation

```bash
npm install moon
```

## Usage

```js
import { getMoonPhase } from "moon";

console.log(getMoonPhase(new Date()));
```
